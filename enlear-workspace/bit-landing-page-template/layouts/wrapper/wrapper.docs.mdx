import { Wrapper } from './wrapper';

A component that renders a parent if a condition is present, else returns the children

### Component usage

```js
<Wrapper
  condition={true}
  wrapper={(children) => {
    <div>{children}</div>;
  }}
>
  {any content}
</Wrapper>
```

### Rendering Hello World Wrapped In `div`!

```js live
<Wrapper condition={true} wrapper={(children) => <div>{children}</div>}>
  Hello World
</Wrapper>
```
